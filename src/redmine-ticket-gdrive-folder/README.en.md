# Redmine ticket Google Drive folder

[English](README.en.md) - [日本語](README.ja.md) - [Tiếng Việt](README.md)

Bookmarklet Name: **RedmineGDriveFolder**

This is a bookmarklet used to generate Google Drive folder names from Redmine ticket information. The bookmarklet copies a properly formatted folder name to the clipboard for creating Google Drive folders for tickets.

The result saved to the clipboard has the following format:
```
21814.Feature.This is ticket title
```

Format: `<ticket number>.<task type>.<ticket title>`

The code is stored in the file [bookmarklet.js](bookmarklet.js)

For installation and usage instructions of the bookmarklet, see details [here](../../README.en.md).
