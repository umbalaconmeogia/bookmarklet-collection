# Redmine ticket Google Drive folder

[English](README.en.md) - [日本語](README.ja.md) - [Tiếng Việt](README.md)

Bookmarklet Name: **RedmineGDriveFolder**

Đây là bookmarklet dùng để tạo tên thư mục Google Drive từ thông tin Redmine ticket. Bookmarklet sẽ copy vào clipboard tên thư mục theo định dạng phù hợp để tạo thư mục Google Drive cho ticket.

Kết quả được lưu vào clipboard có dạng như sau:
```
21814.Feature.This is ticket title
```

Định dạng: `<số ticket>.<loại task>.<tiêu đề ticket>`

Code được lưu trong file [bookmarklet.js](bookmarklet.js)

Về cách cài đặt và sử dụng bookmarklet, xem chi tiết [tại đây](../../README.md).
